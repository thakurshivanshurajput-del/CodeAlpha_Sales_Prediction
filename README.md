# CodeAlpha Internship: Sales Prediction Using Python

This repository contains the complete project submission for **Task 4: Sales Prediction** as part of the **Data Science Internship at CodeAlpha**. The project implements a Multiple Linear Regression framework to evaluate and forecast how budget allocations across different advertising platforms affect total sales returns.

## 📊 Core Objectives
* Clean and transform marketing channel media datasets.
* Extract linear correlations between specific platform channels and product revenue.
* Train an optimized machine learning regression model to accurately predict sales metrics.
* Establish data-driven insights to refine corporate marketing budgets.

## 🛠️ Tech Stack & Key Modules
- **Language:** Python 3.x
- **Libraries:** Pandas, NumPy, Scikit-Learn (Model Building), Matplotlib & Seaborn (Data Visualization).

## 📈 Executive Performance Insights
1. **Model Strength:** The predictive engine achieved an **R² Score of 0.8994**, meaning that roughly **90%** of all sales variations are directly explained by shifts in advertising spend.
2. **Channel Performance Matrix:**
   * **Radio ($\beta = 0.1892$):** Offers the highest unit efficiency. Every additional \$1,000 invested yields a boost of **189.2 units** sold.
   * **TV ($\beta = 0.0447$):** Serves as the primary driver for overall market volume, maintaining a high positive correlation (**0.78**) with total sales.
   * **Newspaper ($\beta = 0.0028$):** Shows a statistically negligible relationship, generating practically zero return on investment.
